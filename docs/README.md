# Documentation & References

This directory acts as the central knowledge base for the project, containing all static reference documents required for system architecture and theoretical justification.

**Contents:**
* **Reference Papers:** Academic literature and technical articles detailing acoustic feature extraction, audio event detection topologies, and TinyML optimization techniques.
* **Datasheets:** Official specifications for critical hardware components (STM32 Cortex-M MCU, I2S MEMS Microphone, Haptic Actuators, PMIC/LDOs).
* **System Constraints:** Power budget estimations and form-factor requirements for the wearable design.

**Note:** Do not store compiled binaries, IDE installers, or generated code in this directory.
